General Instructions

This is a web scraper building application, designed to collect data from websites to facilitate automation of the investigation process. The URL input near the top of this page will accept the starting page for the agent. After entering the starting URL, clicking 'Go To Page' will populate the frame below with the target web page. The HTML structure can now be viewed by using the browser developer tools. The portion of the HTML structure that pertains to the target site will be listed as descendants of the #document element. The developer tools can be accessed by hitting F12. Google Chrome is the recommended browser for use with this application. Once developer tools are displayed, click on the "Elements" header at the top of the window. Now right-clicking on any element within the page shown in the frame, and subsequently selecting "Inspect" at the bottom of the list will cause the corresponding element to be highlighted in the developer tools window.

The manner by which this application identifies HTML elements is through a combination of CSS selectors and jQuery selectors. For convenience, a list of these selectors will be displayed upon clicking 'Selectors'. In order to ensure consistent data scraping, an desired target element must either be consistent in its position in the HTML document, or have a unique selector value. This value can be typed or copy pasted from the developer tools window to populate the action with the corresponding element.

The purpose of using these tools is to build an action list, which will be executed synchronously on the server at run time. The action list should follow the simulated user actions that are tested in the frame window. Testing data can added by clicking 'Add Collection'.
